## [v1.0.0] - 2026-02-08

**Link Pocket의 첫 번째 정식 버전 릴리즈입니다!**
기존의 링크 관리 기능을 강화하고, 사용자 경험(UX)을 대폭 개선한 새로운 UI를 선보입니다.

### 새로운 기능
- **프로필/헤더 통합 UI**: 프로필 버튼을 상단 타이틀 바에 통합하여 화면 공간 활용도를 높이고 디자인 완성도를 향상시켰습니다.
- **멀티 계정 지원**: 프로필 팝업 메뉴에 `다른 계정 추가` 기능을 도입하여, 여러 구글 계정을 손쉽게 전환하며 사용할 수 있습니다.
- **폴더 라이브 정렬**: 사이드바에서 드래그 앤 드롭으로 폴더 순서를 실시간으로 변경하고 저장하는 기능을 추가했습니다.
- **게스트/유저 모드 전환**: 로그인 상태와 비로그인(로컬) 상태를 명확히 구분하고 자연스럽게 전환되도록 개선했습니다.

### 개선 사항
- **프로필 팝업 리뉴얼**: 구글 스타일의 카드형 팝업으로 디자인을 전면 개편했습니다. 
    - 불필요한 여백 제거 및 버튼 레이아웃 최적화.
    - 아이콘(`SVG`)과 텍스트의 정렬을 개선하고 시각적 클리핑 문제를 해결했습니다.
- **사이드바 사용성 개선**:
    - `순서 변경` 버튼을 사이드바 하단으로 이동하여 접근성을 높였습니다.
    - `새 폴더` 버튼과 `순서` 버튼의 비율을(2:1) 조정하여 시각적 균형을 맞췄습니다.
- **이미지 최적화**: 카드 리스트 로딩 시 상위 2개 이미지는 `eager`, 나머지는 `lazy` 로딩을 적용하여 체감 속도를 높였습니다.

### 버그 수정
- **데이터 로딩 오류 해결**: 계정 로그아웃/전환 시 기존 데이터베이스 리스너가 해제되지 않아 발생하던 데이터 표시 오류를 수정했습니다.
- **Null Reference 에러 수정**: UI 요소가 렌더링되기 전이나 삭제된 요소를 참조할 때 발생하던 스크립트 오류를 방어 코드로 해결했습니다.
- **플레이스홀더 이미지 교체**: 서비스가 중단된 `via.placeholder.com` 대신 `placehold.co`를 적용하여 엑박(이미지 깨짐) 현상을 해결했습니다.
