# Link Pocket 🔗

Link Pocket은 웹 상의 유용한 링크들을 쉽고 간편하게 저장하고 관리할 수 있는 웹 애플리케이션입니다.
URL만 입력하면 제목, 이미지, 설명을 자동으로 불러와 카드 형태로 아름답게 정리해 줍니다.

---

## ✨ 주요 기능 (Features)

### 1. 📂 스마트한 링크 저장
- URL을 붙여넣기만 하면 **Open Graph(OG) 태그**를 분석하여 썸네일, 제목, 요약을 자동으로 저장합니다.
- YouTube, Music 등 링크의 타입을 자동으로 인식하여 필터링 태그를 제공합니다.

### 2. 🗂 폴더 관리 시스템
- 주제별로 **폴더**를 생성하여 링크를 체계적으로 분류할 수 있습니다.
- **드래그 앤 드롭** 기능을 지원하여 폴더의 순서를 자유롭게 변경할 수 있습니다.
- 링크를 드래그하여 다른 폴더로 손쉽게 이동시킬 수 있습니다.

### 3. 👤 계정 및 데이터 동기화
- **Google 로그인**을 지원하여 언제 어디서나 데이터를 동기화할 수 있습니다.
- **다중 계정 지원**: 프로필 팝업에서 손쉽게 다른 구글 계정을 추가하고 전환할 수 있습니다.
- **게스트 모드**: 로그인 없이도 로컬 저장소(Local Storage)를 활용하여 즉시 사용 가능합니다.

### 4. 🎨 직관적인 UI/UX
- 구글 머티리얼 디자인 스타일의 깔끔하고 현대적인 인터페이스.
- 반응형 디자인으로 PC와 모바일 환경 모두 최적화되어 있습니다.
- 다크 모드(시스템 설정에 따라)와 시각적 균형을 고려한 레이아웃.

## 🚀 사용 가이드 (Getting Started)

### 1️⃣ 시작하기
- 앱에 접속하면 바로 **게스트 모드**로 시작할 수 있습니다.
- 데이터를 안전하게 보관하려면 우측 상단의 **프로필 아이콘**을 눌러 **로그인**하세요.

### 2️⃣ 링크 추가하기
1. 상단의 입력창(`URL을 입력하세요`)에 저장하고 싶은 웹사이트 주소를 붙여넣습니다.
2. **저장** 버튼을 클릭합니다.
3. 잠시 후 썸네일과 함께 링크 카드가 생성됩니다.

### 3️⃣ 폴더 정리하기
1. 좌측 상단의 **☰ 메뉴** 버튼을 눌러 사이드바를 엽니다.
2. 하단의 **+ 새 폴더 만들기** 버튼으로 폴더를 추가합니다.
3. **순서** 버튼을 누르면 편집 모드가 되며, 폴더를 드래그하여 순서를 바꿀 수 있습니다.

### 4️⃣ 링크 관리하기
- 저장된 링크 카드의 **점 3개(⋮)** 메뉴를 눌러 제목 수정, 폴더 이동, 삭제 등을 할 수 있습니다.
- 중요하거나 나중에 볼 링크는 ⭐️ 별표를 눌러 **즐겨찾기** 할 수 있습니다.
